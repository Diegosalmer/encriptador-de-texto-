# Encriptador de Texto Simple y Eficiente

## Descripción
Este proyecto es una aplicación web que permite a los usuarios encriptar y desencriptar texto de manera eficiente. Está diseñado para quienes necesitan proteger mensajes o información básica, convirtiendo el texto claro en un formato cifrado. Aunque el cifrado utilizado es menos complejo que otros métodos avanzados, sigue siendo robusto y efectivo para asegurar la información.

## Características
- **Encriptación simple pero segura:** Emplea un método de cifrado menos complejo, pero efectivo para proteger la información.
- **Desencriptación directa:** Facilita la recuperación del texto original utilizando la clave correspondiente.
- **Interfaz amigable:** Diseño intuitivo y responsive, que asegura una experiencia de usuario fluida en cualquier dispositivo.
- **Funciones de copiado y pegado:** Permite copiar y pegar fácilmente el texto encriptado o desencriptado para su uso en otras aplicaciones.

## Uso
1. Ingresa el texto que deseas encriptar en el campo correspondiente.
2. Haz clic en el botón "Encriptar" para convertir el texto en un formato cifrado.
3. Para desencriptar, ingresa el texto cifrado y la clave correspondiente, luego haz clic en "Desencriptar".
4. Utiliza las funciones de copiado y pegado para transferir el texto encriptado o desencriptado a otras aplicaciones según sea necesario.

## Requisitos
- Navegador web compatible con HTML5 y CSS3.
